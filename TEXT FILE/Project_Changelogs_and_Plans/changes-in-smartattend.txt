# SmartAttend — Advanced Attendance Management System

SmartAttend is a professional, full-stack attendance management solution designed for universities. It features real-time facial recognition, automated email reporting, and a robust Google Sheets backend.

## 🚀 Recent Major Updates & Change Log

### 1. Full Backend Migration to Google Cloud
**Objective**: Transition from unstable `localStorage` to a unified Google Apps Script API.

*   **Centralized Configuration (`backend/config.js`)**: Created a single source of truth for the `SCRIPT_URL`. All pages now reference this central file.
*   **HTML Portals Modernization**: Updated the following files to include `config.js` and use dynamic backend routing:
    - `Login.html` & `student_login.html`

## 🛠 Technical Architecture

- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6), JQuery.
- **Face AI**: `face-api.js` (weights loaded via CDN).
- **Backend**: Google Apps Script (REST API).
- **Database**: Google Sheets (Multi-tab architecture).
- **Communication**: Google MailApp for OTP, Absence Alerts, and Parent Notifications.

## 📋 How to Sync New Changes

1.  **Script Update**: Copy code from `SmartAttend_AppScript_FINAL.gs` to your Google Script Editor and click **"New Deployment"**.
2.  **Config Update**: Update the `SCRIPT_URL` in `backend/config.js` with your latest deployment link.
3.  **Sheet Setup**: Ensure your Student Sheet has a column named `Parent_Email`.

### 14. Mobile App Deployment via **Netlify**
**Objective**: Document the official method for deploying SmartAttend as an installable Progressive Web App (PWA).

    1. Drag and drop the project folder into `drop.netlify.com`.
    2. Share the generated HTTPS link.
    3. When users visit the link on mobile, the secure connection allows the browser to read `manifest.json` and prompt the **"Add to Home Screen"** installation.
